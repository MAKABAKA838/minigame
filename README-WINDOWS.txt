Bottle Hero - Windows Playtest

How to play:
1. Unzip this folder.
2. Double-click start-game-windows.bat. Do not open index.html directly.
3. Your browser should open automatically.
4. Keep the black command window open while playing.
5. Close the command window when finished.

If Windows blocks the script, right-click start-game-windows.bat and choose Run.
If the browser does not open, copy this address into your browser after the black window starts:
http://127.0.0.1:4173/
